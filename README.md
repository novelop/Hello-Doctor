# Hello-Doctor

### 【의사 협진 웹 서비스】
- 대형병원에서 원내 서로 다른 전공의 의료진 들이 전인적환자 치료를 목표로 효과적인 환자 정보 공유, 빠르고 효율적인 진료, 환자 의료 정보의 보안에 집중하여 협진 할 수 있는 EMR(Electronic Medical Record)과 PMS(Project Management System)가 결합된 의사 협진 웹 서비스를 개발하였습니다.
